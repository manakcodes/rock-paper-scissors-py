## ğŸ› ï¸ Project Setup

### 1. Navigate To The Directory Where You Want To Clone The Project Repository (example: `Desktop`)

```bash
cd ~/Desktop
```

### 2. Clone the Project Repository

```bash
git clone https://github.com/manakcodes/rock-paper-scissors-py.git
```

### 3. Navigate To The Project Repository

```bash
cd path-to/rock-paper-scissors-py/
```

### 4.Install Dependencies

```bash
python3 -m pip install -r requirements.txt
```

### 5. Run The Following Command (to activate the virtual environment `(venv)`)

mac/linux

```bash
source venv/bin/activate
```

windows

```bash
venv\Scripts\activate
```

### 6. Run the project

```bash
python3 Main.py
```
